# odin-recipes
Introduction

This project is part of the Foundations Course and focuses on applying core HTML concepts by building a simple recipe website. The site consists of a main index page that links to multiple individual recipe pages.

The primary objective of this project is to strengthen fundamental HTML skills, including structuring content, creating links, organizing files, and building a clean semantic layout. While the visual design is minimal at this stage, the emphasis is on mastering structure and markup before progressing to CSS styling in future lessons.

By completing this project, I demonstrate the ability to:

Structure a multi-page website using HTML

Create and manage internal links between pages

Organize project files effectively

Use Git and GitHub for version control

Follow a structured development workflow

This project serves as a foundational step toward building more advanced, fully styled, and interactive web applications.
